# 🛡️ 제주 가디언 (Jeju Guardian)

`제주 가디언`은 제주도의 고질적인 문제인 **해양 안보(불법 어선)**와 **환경 오염(바다 쓰레기)**을 하나의 솔루션으로 해결하고자 합니다.

---

### 🤔 어떤 문제를 해결하나요?

지금까지 불법 어선을 단속하는 '해경 함정'과 바다 쓰레기를 치우는 '정화 선박'은 따로 운영되어야 했습니다. 이는 막대한 예산과 인력이 중복으로 투입되는 비효율적인 방식이었습니다.

### ✨ 저희의 핵심 아이디어는 이렇습니다

저희는 **'적을 포위하는 군사 작전'**과 **'쓰레기를 그물로 모으는 환경 정화'**가 본질적으로는 동일한 '탐지(Detection)'와 '제어(Control)' 문제라는 점에 주목했습니다.

> **하드웨어는 그대로 두고, 소프트웨어(AI의 목표)만 바꾸면 두 가지 임무를 모두 수행할 수 있습니다.**

`제주 가디언`은 똑똑한 **자율 무인 수상정(USV) 함대**입니다. 평소에는 바다를 청소하다가(**환경 모드** 🏞️), 불법 어선이 나타나면 즉시 **방어 모드**🛡️로 전환하여 경로를 차단하고 포위합니다.

### ⚙️ 어떻게 작동하나요?

AWS 클라우드 기술을 이용해 하늘(위성)부터 바다(로봇)까지 이어지는 완벽한 자율 관제 시스템을 구축합니다.

1.  🛰️ **감시:** **AWS 위성 수신 기술**로 제주 해역을 24시간 감시합니다.
2.  🧠 **분석:** **AI(YOLOv8)**가 위성 사진을 분석해 불법 어선과 쓰레기의 위치를 정확히 찾아냅니다.
3.  👆 **명령:** 관제사가 웹 대시보드에서 상황을 확인하고 출동 명령을 내립니다.
4.  🚤 **대응:** 명령을 받은 USV 함대가 **강화학습 AI**를 통해 스스로 최적의 경로와 대형을 짜서 임무를 수행합니다.

### 🛠️ 주요 기술 스택

*   **Cloud:** AWS (Ground Station, S3, SageMaker, IoT Core 등)
*   **AI / ML:** 다중 에이전트 강화학습 (MARL), YOLOv8
*   **Edge Computing:** NVIDIA Jetson Orin Nano
*   **Digital Twin:** Unity 3D
